# Synthesis

## When to use

Use this skill when multiple inputs need to be combined into a single coherent view.

## What it does

- Consolidates repeated facts.
- Preserves important distinctions.
- Produces a decision-ready summary.

## Instructions

- Keep the original meaning intact.
- Surface conflicts instead of smoothing them away.
- State conclusions and the evidence behind them.

## Guardrails

- Do not over-compress important nuance.
- Do not blur recommendations into facts.
