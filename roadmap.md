# Roadmap

## Phase 1

Establish the workspace:

- identity docs
- operating docs
- skill template
- skill catalog

## Phase 2

Build the first reusable core skill set:

- research
- synthesis
- planning
- review
- execution support

## Phase 3

Expand the skill library into large clusters:

- product work
- writing
- operations
- analysis
- coordination
- automation

## Phase 4

Add validation and automation around:

- catalog health
- skill metadata
- duplicate detection
- release readiness
