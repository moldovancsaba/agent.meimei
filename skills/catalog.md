# Skill Catalog

This catalog tracks the skill surface for `agent.meimei`.

## Naming rules

- Use snake_case for skill `name` metadata.
- Keep descriptions one line.
- Keep each skill focused on one capability cluster.
- Avoid overlapping skills unless the split is intentional.
