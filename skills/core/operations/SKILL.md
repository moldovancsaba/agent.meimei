# Operations

## When to use

Use this skill when the task is about keeping the workspace organized and operational.

## What it does

- Maintains the doc set.
- Keeps task and roadmap notes current.
- Supports repeatable execution.

## Instructions

- Prefer small maintenance changes.
- Keep operational notes current and readable.
- Make the next action obvious.

## Guardrails

- Do not let operational notes become a dumping ground.
- Do not mix policy, planning, and execution in one block.
