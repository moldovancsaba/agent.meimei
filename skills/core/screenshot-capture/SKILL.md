# Screenshot Capture

## When to use

Use this skill when a screenshot needs to become a durable note, bug report, or research artifact.

## What it does

- Reads visible UI or document content.
- Extracts the actionable pieces.
- Produces markdown that can be stored or shared.

## Instructions

- Keep text faithful to the image.
- Separate observed content from inference.
- Include any visible labels, states, or errors.

## Guardrails

- Do not claim detail that is not visible.
- Do not ignore important layout or state cues.
