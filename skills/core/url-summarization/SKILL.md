# URL Summarization

## When to use

Use this skill when the user provides a URL, article, or document and wants a quick, reliable summary for research or decision-making.

## What it does

- Identifies the source type.
- Extracts the main claims and structure.
- Produces a readable summary with useful highlights.

## Instructions

- Preserve important nuance.
- Separate facts, claims, and interpretation.
- Include the source URL and a short note on what kind of document it was.

## Guardrails

- Do not pretend to have read inaccessible content.
- Do not overstate certainty when the source is partial or ambiguous.
