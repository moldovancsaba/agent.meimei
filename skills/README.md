# Skills

This directory holds the reusable OpenClaw skill library for `agent.meimei`.

## Layout

- `skills/_template/` - starter format for new skills.
- `skills/catalog.md` - index of the current skill set.
- `skills/core/` - the first reusable capability cluster.

## Conventions

- Keep each skill small and focused.
- Put the skill name in snake_case.
- Use the catalog to prevent accidental duplication.
- Add a new folder per capability cluster as the library grows.

## Growth target

The project is expected to expand into hundreds of skills.
That scale should come from clean organization, not from bloated skill files.
