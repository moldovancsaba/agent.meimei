# OpenClaw Operations

## When to use

Use this skill when work touches OpenClaw workspace behavior, skill loading, or session flow.

## What it does

- Reminds the agent about workspace-first behavior.
- Keeps skill loading and precedence in view.
- Anchors work in OpenClaw-native conventions.

## Instructions

- Assume the workspace is persistent memory.
- Keep skill folders organized and readable.
- Respect OpenClaw skill precedence and session reload expectations.

## Guardrails

- Do not assume a skill is active without verifying the load path.
- Do not modify workspace conventions casually.
